# 💸 FinFlow

> Track smart. Spend wisely.

FinFlow is a modern **Expense Tracker Web App** built with **HTML, CSS, JavaScript, Firebase Authentication, and Firestore**.

It helps users manage their income and expenses, visualize spending habits, and keep track of their finances with a clean and aesthetic interface inspired by modern Gen Z apps.

---

## ✨ Features

### 🔐 Authentication
- User Sign Up
- User Login
- User Logout
- Firebase Authentication (Email & Password)

### 💰 Expense Management
- Add Income & Expenses
- Categorize transactions
- Add notes and dates
- Real-time balance calculation

### 📊 Analytics
- Balance summary cards
- Income & Expense tracking
- Pie Chart for expense categories
- Monthly Bar Chart

### 🎨 UI / UX
- Aesthetic modern design
- Dark Mode 🌙
- Responsive layout
- Smooth interactions

### ☁️ Database
- Firebase Firestore integration
- User-specific transaction storage
- Secure cloud database

## ⚙️ Firebase Setup

1. Create a Firebase Project.
2. Enable **Authentication → Email/Password**.
3. Create **Cloud Firestore Database**.
4. Add your Firebase configuration to:

```javascript
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
```
